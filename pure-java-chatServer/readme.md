# Pure-java-chatServer


## 주요 요건 정의
- 다중 클라이언트 접속 및 텍스트 기반 채팅 처리
- 특정 클라이언트로부터 수신한 메시지 전체 사용자에게 브로드캐스팅


## 개발 관련 학습 목표
| 항목               | 설명                                            |
|------------------|-----------------------------------------------|
| 네트워크 기초 학습   | TCP/IP, Socket, ServerSocket 등 저수준 개념         |
| 멀티스레딩 이해     | 클라이언트별로 스레드 분리 또는 스레드풀로 관리                    |
| 리소스 관리 연습    | Input/Output Stream, 연결 종료 처리, 예외 처리 등을 직접 구현 |
| 성능 병목 이해     | **blocking I/O** 기반에서 다수의 연결을 처리할 때의 한계 체험    |
| 프레임워크 없이 로직 구성 | 메시지 브로드캐스트, 사용자 목록, 명령어 처리 등을 직접 구현           |
